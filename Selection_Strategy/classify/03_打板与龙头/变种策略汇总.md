# 变种策略汇总

## 一、本族策略主干思想

本族策略以「围绕涨停、首板、连板与龙头股进行短线交易」为核心逻辑：利用涨停信号、集合竞价、首板低开、弱转强、龙虎榜等捕捉情绪与资金驱动的短线机会。

## 二、变种维度归纳

本族内各子策略主要在以下维度上形成变种（可对照下表查看每个策略的具体实现）：

- 介入点：首板低开 / 集合竞价 / 一进二 / 连板龙头 / 弱转强 / 龙虎榜
- 卖出：11:25止盈 / 14:30清仓 / 涨停打开卖出 / 固定止损
- 过滤：不追高 / 不接高位连板 / 剔除新股
