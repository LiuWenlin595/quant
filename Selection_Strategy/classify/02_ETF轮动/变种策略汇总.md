# 变种策略汇总

## 一、本族策略主干思想

本族策略以「在ETF池内按动量等指标轮动持仓」为核心逻辑：在若干宽基/行业/跨境ETF中，选取近期动量最强（或结合其他指标）的标的持有，定期再选、强者恒强。

## 二、变种维度归纳

本族内各子策略主要在以下维度上形成变种（可对照下表查看每个策略的具体实现）：

- 动量定义：N日收益 / 乖离率动量 / R-squared
- 择时：RSRS / 北向 / MA乖离 / 盘中止损
- 标的：宽基 / 行业 / 多品种 / 核心资产
- 优化：EPO / 相关性最小 / 波动率过滤 / 股债平衡
