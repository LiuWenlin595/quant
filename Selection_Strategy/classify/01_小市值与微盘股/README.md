# 小市值与微盘股

## 策略主体方法

以「定期买入市值最小的一批股票并持有、再调仓」为核心。变种包括：基本面/盈利过滤、股息率、国九条与分红要求、1月4月空仓、机器学习或保形回归选股、微盘400/微盘三正、止损止盈与动态仓位等。
